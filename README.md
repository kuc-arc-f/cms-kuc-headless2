### Setup

npm install

***
### Setup , etc

* .env

MY_SITE_NAME: 

```
MY_SITE_NAME=sample CMS
```

MY_JSON_URL: if change json URL,

```
MY_JSON_URL=https://sample.netlify.app/files/cms-kuc-headless2/cms.json
```

***
### start server
* Start :

yarn dev

* build:

yarn deploy
